What is the File System?

To define the File System it is best to start with save data. Save data is a feature that we've come to expect from computers. It allows us to keep data around for a longer period of time than would otherwise be possible. In a world void of save data our data would be temporary; you may have some experience of this if you've ever lost data. So I hope we can agree that save data is a feature that is worth pursuing. However, in order to provide that feature we need to solve a number of problems. Some of these problems relate to hardware and some relate to software. The most successful solution to the software related problems is built around a core concept we call a file. This is why we call it 'The File System'. Hence the File System is software that is necessary if we want to save data. For the most part we trust it, because it has been reliable time and time again. Data corruption isn't something we experience often because the File System ensures our data

The File System wouldn't exist if we didn't have a need to save data. It is impossible to define the File System without talking about save data. To define the File System let's begin by thinking about how computers save data. For a computer to provide this saving feature it must overcome a number of problems. Some are hardware related and some are software related. If we solve all the software related problems we will end up with a save system, which comprises of those solutions.

Lets take a deeper look into how a computer goes about saving data. For us to save data on a computer, hardware problems and software problems need to be solved. The software problems require a combination of solutions, and the File System is one such combination. In order to save data on your computer a number of problems need to be solved. Some of those problems are hardware related and some of those problems are software related. The File System is a bunch of solutions to the software related problems we need to overcome in order to save data.
